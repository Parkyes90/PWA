# IPManagementSystem

This is a WebAPP for IP Management 

made by Django && React 

## Environment
Python3.7

Node 10.15.3

## Install

git clone https://github.com/Parkyes90/IPManagementSystem

```
cd IPManagementSystem
pipenv shell && pipenv install
python manage.py runserver
```
```
cd static && npm i
npm run start
```
